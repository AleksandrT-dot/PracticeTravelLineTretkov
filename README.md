# TravelLine Tech - Управление контентом

CMS для управления контентом сайта TravelLine Tech.

## Возможности

- Редактирование Hero-заголовка
- Управление преимуществами
- Управление командой (добавление/удаление/редактирование)
- Управление направлениями
- Управление вакансиями
- Сохранение в JSON-файл
- Сброс к исходным данным

## Технологии

- Node.js + Express
- HTML + CSS + JavaScript
- JSON-хранилище
- GSAP + Swiper.js

## Установка и запуск

```bash
# 1. Клонируйте репозиторий
git clone <url>
cd TravelLine

# 2. Установите зависимости
npm install

# 3. Запустите сервер
node server.js
